# ადმინისტრატორის ბრძანებები (Terminal Commands)

აქ მოცემულია ყველა საჭირო ბრძანება პროექტის სამართავად.

## 1. სერვერის ჩართვა

პროექტის გასაშვებად გამოიყენეთ ერთ-ერთი ბრძანება:

```bash
node server.js
```

ან

```bash
npm start
```

## 2. პაროლის შეცვლა

მომხმარებლის პაროლის შესაცვლელად (ჰეშირების ჩათვლით):

```bash
node scripts/change_password.js <ელ.ფოსტა> <ახალი_პაროლი>
```

**მაგალითი:**

```bash
node scripts/change_password.js gkhosroshvili93@gmail.com AxaliParoli123!
```

## 3. სატესტო მომხმარებლების გენერაცია

თუ გსურთ 30 ახალი მომხმარებლის დამატება (წინა მონაცემები შენარჩუნდება/განახლდება):

```bash
node scripts/generate_users.js
```

## 4. ბიბლიოთეკების დაყენება

თუ პროექტი ახალ კომპიუტერზე გადაიტანეთ:

```bash
npm install
```
