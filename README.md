# Node.js рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ (2FA рЃЏрЃ«рЃљрЃарЃЊрЃљрЃГрЃћрЃарЃўрЃЌ)

рЃћрЃА рЃљрЃарЃўрЃА рЃАрЃарЃБрЃџрЃДрЃЮрЃцрЃўрЃџрЃў рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃўрЃА рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ, рЃЊрЃљрЃгрЃћрЃарЃўрЃџрЃў **Node.js**-рЃќрЃћ. рЃърЃарЃЮрЃћрЃЦрЃбрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљрЃА, рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљрЃА (Login) рЃЊрЃљ рЃЮрЃарЃцрЃљрЃЦрЃбрЃЮрЃарЃўрЃљрЃю рЃљрЃБрЃЌрЃћрЃюрЃбрЃўрЃцрЃўрЃЎрЃљрЃфрЃўрЃљрЃА (2FA) Google Authenticator-рЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

## ­Ъџђ рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў

- **рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃљ рЃћрЃџ.рЃцрЃЮрЃАрЃбрЃўрЃЌ рЃЊрЃљ рЃърЃљрЃарЃЮрЃџрЃўрЃЌ.
- **рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃЋрЃљрЃџрЃўрЃЊрЃљрЃфрЃўрЃљ**: рЃЏрЃЮрЃўрЃЌрЃ«рЃЮрЃЋрЃА рЃЏрЃўрЃюрЃўрЃЏрЃБрЃЏ 8 рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃА, рЃфрЃўрЃцрЃарЃћрЃЉрЃА рЃЊрЃљ рЃљрЃАрЃЮрЃћрЃЉрЃА.
- **2FA-рЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ**: QR рЃЎрЃЮрЃЊрЃўрЃА рЃњрЃћрЃюрЃћрЃарЃљрЃфрЃўрЃљ Google Authenticator-рЃўрЃАрЃЌрЃЋрЃўрЃА.
- **рЃљрЃЋрЃбрЃЮрЃарЃўрЃќрЃљрЃфрЃўрЃљ (Login)**: рЃерЃћрЃАрЃЋрЃџрЃљ рЃърЃљрЃарЃЮрЃџрЃўрЃЌ рЃЊрЃљ рЃћрЃарЃЌрЃ»рЃћрЃарЃљрЃЊрЃў рЃЎрЃЮрЃЊрЃўрЃЌ.
- **рЃАрЃћрЃАрЃўрЃћрЃЉрЃў**: рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃАрЃћрЃАрЃўрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљ.

## ­ЪЏа рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃБрЃџрЃў рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃў

- **Node.js & Express**: рЃАрЃћрЃарЃЋрЃћрЃарЃБрЃџрЃў рЃЏрЃ«рЃљрЃарЃћ.
- **EJS**: рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃЏрЃ«рЃљрЃарЃћ (HTML рЃерЃљрЃЉрЃџрЃЮрЃюрЃћрЃЉрЃў).
- **otplib & qrcode**: 2FA рЃцрЃБрЃюрЃЦрЃфрЃўрЃЮрЃюрЃљрЃџрЃў.
- **express-session**: рЃАрЃћрЃАрЃўрЃћрЃЉрЃўрЃА рЃЏрЃћрЃюрЃћрЃ»рЃЏрЃћрЃюрЃбрЃў.

## ­ЪЊд рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ рЃЊрЃљ рЃњрЃљрЃерЃЋрЃћрЃЉрЃљ

1.  **рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃЊрЃљрЃДрЃћрЃюрЃћрЃЉрЃљ**:
    рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃў рЃЊрЃљ рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ:

    ```bash
    npm install
    ```

2.  **рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃА рЃЕрЃљрЃарЃЌрЃЋрЃљ**:

    ```bash
    node server.js
    ```

    рЃљрЃю

    ```bash
    npm start
    ```

3.  **рЃАрЃљрЃўрЃбрЃўрЃА рЃњрЃљрЃ«рЃАрЃюрЃљ**:
    рЃњрЃљрЃ«рЃАрЃћрЃюрЃўрЃЌ рЃЉрЃарЃљрЃБрЃќрЃћрЃарЃў рЃЏрЃўрЃАрЃљрЃЏрЃљрЃарЃЌрЃќрЃћ: [http://localhost:3000](http://localhost:3000)

## ­ЪЊѓ рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљ

- **`server.js`**: рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА "рЃбрЃЋрЃўрЃюрЃў". рЃљрЃЦ рЃгрЃћрЃарЃўрЃљ рЃЏрЃЌрЃћрЃџрЃў рЃџрЃЮрЃњрЃўрЃЎрЃљ (рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃљ, рЃџрЃЮрЃњрЃўрЃюрЃў, 2FA рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ).
- **`views/`**: рЃљрЃЦ рЃљрЃарЃўрЃА рЃАрЃљрЃўрЃбрЃўрЃА рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃарЃў рЃцрЃљрЃўрЃџрЃћрЃЉрЃў (рЃњрЃЋрЃћрЃарЃЊрЃћрЃЉрЃў).
  - `login.ejs`: рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў.
  - `register.ejs`: рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў.
  - `setup-2fa.ejs`: 2FA-рЃА рЃАрЃЎрЃљрЃюрЃўрЃарЃћрЃЉрЃўрЃА рЃњрЃЋрЃћрЃарЃЊрЃў.
  - `dashboard.ejs`: рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃњрЃЋрЃћрЃарЃЊрЃў рЃерЃћрЃАрЃЋрЃџрЃўрЃА рЃерЃћрЃЏрЃЊрЃћрЃњ.
- **`package.json`**: рЃърЃарЃЮрЃћрЃЦрЃбрЃўрЃА рЃўрЃюрЃцрЃЮрЃарЃЏрЃљрЃфрЃўрЃљ рЃЊрЃљ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃўрЃА рЃАрЃўрЃљ.
- **`node_modules/`**: рЃЊрЃљрЃЏрЃ«рЃЏрЃљрЃарЃћ рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃћрЃЉрЃў (рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃўрЃЦрЃЏрЃюрЃћрЃЉрЃљ).
