# როგორ ავტვირთოთ პროექტი GitHub-ზე

ვინაიდან თქვენს კომპიუტერში `git` არ არის დაინსტალირებული (ან ტერმინალი ვერ ხედავს მას), ჯერ დაგჭირდებათ მისი დაყენება.

## ნაბიჯი 1: Git-ის ინსტალაცია

1.  გადადით ბმულზე: [git-scm.com/downloads](https://git-scm.com/downloads)
2.  გადმოწერეთ **Windows**-ის ვერსია.
3.  დააინსტალირეთ (ინსტალაციის დროს შეგიძლიათ ყველაფერი დატოვოთ სტანდარტულ მნიშვნელობებზე - "Next", "Next"...).
4.  **მნიშვნელოვანია:** იმისათვის, რომ ტერმინალმა Git დაინახოს, ინსტალაციის შემდეგ შესაძლოა დაგჭირდეთ VS Code-ის ან ტერმინალის გადატვირთვა.

## ნაბიჯი 2: GitHub-ზე რეპოზიტორის შექმნა

1.  შედით თქვენს [GitHub](https://github.com/) პროფილზე.
2.  დააჭირეთ ღილაკს **"New"** (ან `+` ნიშანს ზედა მარჯვენა კუთხეში -> New repository).
3.  **Repository name** ველში ჩაწერეთ პროექტის სახელი (მაგალითად: `my-node-project`).
4.  მონიშნეთ **Public** (საჯარო) ან **Private** (პირადი).
5.  **არ მონიშნოთ** "Add a README file" (რადგან პროექტი უკვე გაქვთ კომპიუტერში).
6.  დააჭირეთ **Create repository**.

## ნაბიჯი 3: პროექტის დაკავშირება და ატვირთვა

GitHub-ზე რეპოზიტორის შექმნის შემდეგ, გამოჩნდება ბრძანებები. თქვენ დაგჭირდებათ შემდეგი ნაბიჯების შესრულება VS Code-ის ტერმინალში (დარწმუნდით, რომ ხართ პროექტის საქაღალდეში `c:\Users\gkhosroshvili\Desktop\node.js`):

1.  **ინიციალიზაცია:**

    ```bash
    git init
    ```

2.  **ფაილების დამატება:**

    ```bash
    git add .
    ```

3.  **შენახვა (Commit):**

    ```bash
    git commit -m "First commit: Project setup"
    ```

4.  **მთავარი ტოტის სახელის შეცვლა (სასურველია, მაგრამ არა სავალდებულო):**

    ```bash
    git branch -M main
    ```

5.  **GitHub-თან დაკავშირება (შეცვალეთ URL თქვენი რეპოზიტორის ბმულით):**

    - დააკოპირეთ ბმული GitHub-დან (ის იქნება `https://github.com/თქვენი_სახელი/რეპოზიტორის_სახელი.git` ფორმატის).
    - ჩაწერეთ ტერმინალში:

    ```bash
    git remote add origin https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
    ```

6.  **ატვირთვა (Push):**
    ```bash
    git push -u origin main
    ```

ამის შემდეგ, GitHub მოგთხოვთ ავტორიზაციას (Sign in with browser), გაიარეთ ავტორიზაცია და პროექტი აიტვირთება.
