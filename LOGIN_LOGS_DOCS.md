# შესვლის ისტორიის (Audit Log) დოკუმენტაცია

ეს ფაილი აღწერს სისტემას, რომელიც იმახსოვრებს ყველა **წარმატებულ** შესვლას.

## 1. ლოგების ფაილი

ყველა მონაცემი ინახება ფაილში: `login_logs.json`

ეს არის `JSON` ფორმატის ფაილი, რომელიც შეიცავს ჩანაწერების სიას.

## 2. რა ინფორმაცია ინახება?

თითოეული შესვლის დროს ვიმახსოვრებთ 4 მონაცემს:

| ველი          | აღწერა                  | მაგალითი                                       |
| :------------ | :---------------------- | :--------------------------------------------- |
| **email**     | მომხმარებლის ელ.ფოსტა   | `admin@test.ge`                                |
| **timestamp** | შესვლის ზუსტი დრო       | `2024-05-20T10:30:15.123Z`                     |
| **ip**        | IP მისამართი            | `::1` (ლოკალური) ან `192.168.1.1`              |
| **device**    | მოწყობილობა და ბრაუზერი | `Mozilla/5.0 (Windows NT 10.0; Win64; x64)...` |

## 3. მაგალითი

```json
[
  {
    "timestamp": "2024-05-20T10:30:15.123Z",
    "email": "testuser1@example.com",
    "ip": "::1",
    "device": "Mozilla/5.0 (Windows NT 10.0; Win64; x64) AppleWebKit/537.36 (KHTML, like Gecko) Chrome/124.0.0.0 Safari/537.36"
  }
]
```

## 4. როგორ ვნახოთ?

გახსენით `login_logs.json` ნებისმიერი ტექსტური რედაქტორით (Notepad, VS Code).
