# პროექტის შემაჯამებელი ანგარიში (ლექტორისთვის)

## 1. პროექტის აღწერა

ეს არის სრულყოფილი ვებ-აპლიკაცია (Full-Stack Web Application), დაწერილი **Node.js**-ის გამოყენებით. პროექტი მოიცავს როგორც მომხმარებლის ინტერფეისს (Frontend), ისე სერვერულ ნაწილს (Backend) და მონაცემთა ბაზას.

## 2. გამოყენებული ტექნოლოგიები

- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas (Cloud Database) - მონაცემების გლობალურად შესანახად.
- **Frontend:** EJS (Templating Engine), HTML, CSS, JavaScript.
- **Hosting/Deployment:** Render (სერვერისთვის) და GitHub (კოდის ვერსიების კონტროლისთვის).

## 3. ძირითადი ფუნქციონალი

1.  **ავტორიზაცია და უსაფრთხოება:**

    - **რეგისტრაცია და ავტორიზაცია:** მომხმარებლებს შეუძლიათ დარეგისტრირდნენ და შევიდნენ სისტემაში.
    - **პაროლების დაშიფვრა:** ვიყენებთ `bcrypt`-ს, რათა პაროლები ბაზაში ღია სახით არ შეინახოს (Hashing).
    - **ორფაქტორიანი ავტორიზაცია (2FA):** Google Authenticator-ის ინტეგრაცია დამატებითი უსაფრთხოებისთვის.
    - **სესიების მართვა:** `express-session`-ის გამოყენება მომხმარებლის სესიის შესანარჩუნებლად.

2.  **მონაცემთა ბაზა (MongoDB Atlas):**

    - პროექტი არ იყენებს ლოკალურ ფაილებს ინფორმაციის შესანახად. ყველა მომხმარებელი, პოსტი და ინფორმაცია ინახება დაცულ ღრუბლოვან (Cloud) ბაზაში, რაც უზრუნველყოფს მონაცემთა საიმედოობას და ხელმისაწვდომობას ნებისმიერი ადგილიდან.

3.  **კიბერუსაფრთხოების ელემენტები:**
    - განხორციელდა "შეღწევადობის ტესტირება" (Penetration Testing) საკუთარ სისტემაზე, რათა გამოგვევლინა და აღგვეკვეთა სუსტი წერტილები.
    - დაცულია ლოკალური კონფიდენციალური ფაილები (`.env`, `users.json`, რეპორტები), რომლებიც არ აიტვირთა საჯარო რეპოზიტორიაში (`.gitignore`-ის საშუალებით).

## 4. დეპლოი (განთავსება)

პროექტი წარმატებით განთავსდა **Render**-ის პლატფორმაზე.

- კოდი ავტომატურად იტვირთება **GitHub**-დან.
- სერვერი უკავშირდება **MongoDB Atlas**-ს.
- შედეგად, აპლიკაცია ხელმისაწვდომია გლობალურად, საჯარო ბმულით.
