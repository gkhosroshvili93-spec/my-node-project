# Bcryptjs: рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃЌрЃљ рЃњрЃќрЃљрЃЏрЃЎрЃЋрЃџрЃћрЃЋрЃў ­ЪЏА№ИЈ

**Bcryptjs** рЃљрЃарЃўрЃА рЃЉрЃўрЃЉрЃџрЃўрЃЮрЃЌрЃћрЃЎрЃљ, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃњрЃЋрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃърЃљрЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃБрЃАрЃљрЃцрЃарЃЌрЃ«рЃЮрЃЊ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃерЃў.

## рЃарЃљрЃбрЃЮрЃЏ рЃњрЃЋрЃГрЃўрЃарЃЊрЃћрЃЉрЃљ?

рЃгрЃљрЃарЃЏрЃЮрЃўрЃЊрЃњрЃўрЃюрЃћрЃЌ, рЃарЃЮрЃЏ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЉрЃљрЃќрЃљрЃерЃў рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃў рЃўрЃюрЃљрЃ«рЃћрЃЉрЃљ рЃарЃЮрЃњрЃЮрЃарЃф рЃбрЃћрЃЦрЃАрЃбрЃў: `"secret123"`.
рЃЌрЃБ рЃЋрЃўрЃюрЃЏрЃћ рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃЉрЃљрЃќрЃљрЃА рЃЏрЃЮрЃўрЃърЃљрЃарЃљрЃЋрЃА, рЃўрЃА рЃДрЃЋрЃћрЃџрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃџрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃА рЃњрЃљрЃўрЃњрЃћрЃЉрЃА!

`bcryptjs` рЃљрЃЏ рЃърЃљрЃарЃЮрЃџрЃА рЃњрЃљрЃарЃЊрЃљрЃЦрЃЏрЃюрЃўрЃА рЃњрЃљрЃБрЃарЃЎрЃЋрЃћрЃЋрЃћрЃџ рЃЎрЃЮрЃЊрЃљрЃЊ (рЃ░рЃћрЃерЃљрЃЊ), рЃЏрЃљрЃњрЃљрЃџрЃўрЃЌрЃљрЃЊ:
`$2b$10$X8f...z9y`

рЃљрЃЏ рЃЎрЃЮрЃЊрЃўрЃЊрЃљрЃю рЃБрЃЎрЃљрЃю рЃърЃљрЃарЃЮрЃџрЃўрЃА (`"secret123"`) рЃљрЃдрЃЊрЃњрЃћрЃюрЃљ **рЃерЃћрЃБрЃФрЃџрЃћрЃЉрЃћрЃџрЃўрЃљ**.

---

## 1. рЃўрЃюрЃАрЃбрЃљрЃџрЃљрЃфрЃўрЃљ

рЃЌрЃЦрЃЋрЃћрЃюрЃА рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃЏрЃўрЃА рЃЊрЃљрЃАрЃљрЃЏрЃљрЃбрЃћрЃЉрЃџрЃљрЃЊ рЃбрЃћрЃарЃЏрЃўрЃюрЃљрЃџрЃерЃў рЃЕрЃљрЃгрЃћрЃарЃћрЃЌ:

```bash
npm install bcryptjs
```

рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў рЃњрЃљрЃЏрЃЮрЃАрЃљрЃДрЃћрЃюрЃћрЃЉрЃџрЃљрЃЊ рЃЎрЃў рЃцрЃљрЃўрЃџрЃўрЃА рЃЌрЃљрЃЋрЃерЃў:

```javascript
const bcrypt = require("bcryptjs");
```

## 2. рЃарЃЮрЃњрЃЮрЃа рЃЏрЃБрЃерЃљрЃЮрЃЉрЃА? (рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃцрЃБрЃюрЃЦрЃфрЃўрЃћрЃЉрЃў)

### A. рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљ (Hashing)

рЃарЃЮрЃфрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃарЃћрЃњрЃўрЃАрЃбрЃарЃўрЃарЃЊрЃћрЃЉрЃљ, рЃЕрЃЋрЃћрЃю рЃЏрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃА рЃЋрЃерЃўрЃцрЃарЃљрЃЋрЃЌ `hash` рЃцрЃБрЃюрЃЦрЃфрЃўрЃўрЃЌ.

```javascript
// рЃърЃљрЃарЃЮрЃџрЃў, рЃарЃЮрЃЏрЃћрЃџрЃўрЃф рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЏрЃљ рЃерЃћрЃўрЃДрЃЋрЃљрЃюрЃљ
const plainPassword = "mySuperPassword123";

// 10 рЃљрЃарЃўрЃА "рЃАрЃўрЃарЃЌрЃБрЃџрЃћ" (Salt Rounds). рЃарЃљрЃф рЃЏрЃљрЃдрЃљрЃџрЃўрЃљ, рЃЏрЃўрЃЌ рЃБрЃцрЃарЃЮ рЃЊрЃљрЃфрЃБрЃџрЃўрЃљ, рЃЏрЃљрЃњрЃарЃљрЃЏ рЃюрЃћрЃџрЃўрЃљ.
// 10 рЃљрЃарЃўрЃА рЃАрЃбрЃљрЃюрЃЊрЃљрЃарЃбрЃБрЃџрЃў рЃЊрЃљ рЃЎрЃљрЃарЃњрЃў рЃљрЃарЃЕрЃћрЃЋрЃљрЃюрЃў.
bcrypt.hash(plainPassword, 10, function (err, hash) {
  if (err) {
    console.error(err);
  } else {
    // hash рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃарЃљрЃдрЃљрЃф рЃљрЃАрЃћрЃЌрЃў: $2b$10$CwTycUXWue0Thq9StjUM0u...
    // рЃАрЃгрЃЮрЃарЃћрЃЊ рЃћрЃА hash рЃБрЃюрЃЊрЃљ рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃЉрЃљрЃќрЃљрЃерЃў!
    console.log("рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃърЃљрЃарЃЮрЃџрЃў:", hash);
  }
});
```

### B. рЃърЃљрЃарЃЮрЃџрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљ (Compare)

рЃарЃЮрЃфрЃљ рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃў рЃфрЃЊрЃўрЃџрЃЮрЃЉрЃА рЃерЃћрЃАрЃЋрЃџрЃљрЃА (Login), рЃЕрЃЋрЃћрЃю рЃБрЃюрЃЊрЃљ рЃерЃћрЃЋрЃљрЃЏрЃЮрЃгрЃЏрЃЮрЃЌ, рЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЏрЃўрЃА рЃЏрЃўрЃћрЃа рЃЕрЃљрЃгрЃћрЃарЃўрЃџрЃў рЃърЃљрЃарЃЮрЃџрЃў рЃЉрЃљрЃќрЃљрЃерЃў рЃерЃћрЃюрЃљрЃ«рЃБрЃџ рЃ░рЃћрЃерЃА.

```javascript
const userPassword = "mySuperPassword123"; // рЃарЃљрЃАрЃљрЃф рЃўрЃБрЃќрЃћрЃарЃў рЃгрЃћрЃарЃА рЃерЃћрЃАрЃЋрЃџрЃўрЃАрЃљрЃА
const dbHash = "$2b$10$CwTycUXWue0Thq9StjUM0u..."; // рЃарЃљрЃф рЃЉрЃљрЃќрЃљрЃерЃў рЃњрЃЋрЃљрЃЦрЃЋрЃА рЃерЃћрЃюрЃљрЃ«рЃБрЃџрЃў

bcrypt.compare(userPassword, dbHash, function (err, isMatch) {
  if (isMatch) {
    console.log("РюЁ рЃърЃљрЃарЃЮрЃџрЃў рЃАрЃгрЃЮрЃарЃўрЃљ! рЃерЃћрЃБрЃерЃЋрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљрЃерЃў.");
  } else {
    console.log("РЮї рЃърЃљрЃарЃЮрЃџрЃў рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ.");
  }
});
```

## рЃарЃљ рЃљрЃарЃўрЃА "Salt"? ­ЪДѓ

`bcrypt` рЃљрЃЋрЃбрЃЮрЃЏрЃљрЃбрЃБрЃарЃљрЃЊ рЃљрЃЏрЃљрЃбрЃћрЃЉрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃАрЃўрЃЏрЃЉрЃЮрЃџрЃЮрЃћрЃЉрЃА (Salt) рЃърЃљрЃарЃЮрЃџрЃА рЃЊрЃљрЃерЃўрЃцрЃЋрЃарЃљрЃЏрЃЊрЃћ.
рЃћрЃА рЃюрЃўрЃерЃюрЃљрЃЋрЃА, рЃарЃЮрЃЏ рЃЌрЃБ рЃЮрЃа рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃљрЃЦрЃЋрЃА **рЃќрЃБрЃАрЃбрЃљрЃЊ рЃћрЃарЃЌрЃў рЃЊрЃљ рЃўрЃњрЃўрЃЋрЃћ рЃърЃљрЃарЃЮрЃџрЃў** (рЃЏрЃљрЃњ: "hello"), рЃЏрЃљрЃЌрЃў рЃЊрЃљрЃерЃўрЃцрЃарЃБрЃџрЃў рЃЎрЃЮрЃЊрЃў (Hash) рЃЏрЃљрЃўрЃюрЃф **рЃњрЃљрЃюрЃАрЃ«рЃЋрЃљрЃЋрЃћрЃЉрЃБрЃџрЃў рЃўрЃЦрЃюрЃћрЃЉрЃљ**.

рЃћрЃА рЃФрЃљрЃџрЃўрЃљрЃю рЃљрЃарЃЌрЃБрЃџрЃћрЃЉрЃА рЃ░рЃљрЃЎрЃћрЃарЃћрЃЉрЃўрЃАрЃЌрЃЋрЃўрЃА рЃърЃљрЃарЃЮрЃџрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃюрЃЮрЃЉрЃљрЃА.

---

## рЃерЃћрЃ»рЃљрЃЏрЃћрЃЉрЃљ

1. **рЃљрЃарЃљрЃАрЃЊрЃарЃЮрЃА** рЃерЃћрЃўрЃюрЃљрЃ«рЃЮрЃЌ рЃърЃљрЃарЃЮрЃџрЃў рЃдрЃўрЃљ рЃбрЃћрЃЦрЃАрЃбрЃљрЃЊ!
2. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `bcrypt.hash` рЃарЃћрЃњрЃўрЃАрЃбрЃарЃљрЃфрЃўрЃўрЃАрЃљрЃА.
3. рЃњрЃљрЃЏрЃЮрЃўрЃДрЃћрЃюрЃћрЃЌ `bcrypt.compare` рЃерЃћрЃАрЃЋрЃџрЃўрЃАрЃљрЃА.
